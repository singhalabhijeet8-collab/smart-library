# Git and GitHub Submission Steps

Run these commands from the project root:

```powershell
git init
git add .
git commit -m "Initial smart library system project"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/smart-library-system.git
git push -u origin main
```

Before pushing, create an empty repository named `smart-library-system` on GitHub.

Useful collaboration evidence for submission:

- GitHub repository URL
- Commit history screenshot
- README file
- Source code folders
- Test report from `mvn test`
- Build artifact from `mvn clean package`

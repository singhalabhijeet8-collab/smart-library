# Smart Library System Diagrams

This folder contains project diagrams for the Smart Library System.

## UML Diagrams

- `uml/use-case-diagram.puml`: shows the main librarian actions.
- `uml/class-diagram.puml`: shows the Swing GUI, Spring Boot layers, entities, services, and repositories.
- `uml/issue-return-sequence.puml`: shows the issue and return book flow.

## Database Diagrams

- `database/er-diagram.puml`: shows the MySQL tables and relationships.
- `database/schema.sql`: SQL script for creating the database tables and adding sample books.

The `.puml` files can be rendered using PlantUML, IntelliJ plugins, VS Code PlantUML extension, or online PlantUML renderers.
